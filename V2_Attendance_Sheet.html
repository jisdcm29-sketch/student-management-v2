반별 기간 출석부

반을 검색해 선택하고 조회 기간을 입력하면 학생 행 / 날짜 열 기준의 출석부를 조회할 수 있다. 이 화면은 엑셀 출력 / 업로드 검증 / 업로드 반영의 기준 화면으로 사용된다.
반별 기간 출석부 데이터를 불러오는 중이다.
업로드 검증 / 오류 로그 안내
업로드 처리 기준 1. 반드시 현재 화면의 반 ID / 기간 조건과 일치해야 한다. 2. 엑셀 출력으로 받은 표준 양식 기준 파일만 검증 또는 반영한다. 3. 업로드 검증은 저장 없이 검사만 수행한다. 4. 업로드 반영은 먼저 검증을 통과한 뒤 실제 저장을 수행한다. 5. 오류 또는 경고 발생 시 상단 알림 박스에 결과가 표시된다.
로그 저장 기준 1. 업로드 검증 실패 / 경고 / 반영 결과는 V2_Logs에 기록된다. 2. detail 원문과 detailJson 구조를 함께 보관한다. 3. 검증과 반영 단계 모두 같은 로그 구조를 재사용한다. 4. source / level / createdAt 기준으로 추적 가능하다.

출석부 표
반을 선택하고 기간을 입력한 뒤 출석부 조회를 실행해라.
안내: 출석 상태는 출석 / 지각 / 결석 / 공결 기준으로 표시된다. 날짜 열은 선택한 기간 전체가 고정 순서로 생성된다. 엑셀 출력은 현재 조회 조건 기준 표준 양식으로 다운로드된다.
